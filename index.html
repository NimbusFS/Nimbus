Nimbus

Nimbus uses FUSE to bring the native file management experience to CloudApp. It allows you to sort, open, upload, delete, and Quicklook your uploaded files.

  • Github
  • Kickstarter

Features

Native File Management

Nimbus mounts your CloudApp files as a drive you can open in Finder. Open, rename, delete, add, and Quicklook your uploaded files!

Fast Access

Nimbus downloads only the files it needs to, saving you bandwidth and time. It also caches frequently used files for speedy access.

Painless Install

Simply download and install. OSXFuse will be installed if you do not already have it.

Sortable

Nimbus creates virtual folders that allow you to view your files by category: Popular, Images, Text, Audio, Video, etc.

How does it work?

Nimbus sits between the CloudApp API and OSXFUSE API, translating Finder actions into CloudApp requests.

Contributing to Nimbus
